I want to enhance my Mission Control dashboard — a space-station-themed 
web app that serves as the front-end for my OpenClaw AI agent system. 
This is an existing project. I'm adding new pixel-art agent sprites and 
improving the Agent Office visual experience.

═══════════════════════════════
WHAT I'M ADDING
═══════════════════════════════

1. NEW AGENT SPRITES
   I have 4 new pixel-art character sprite sheets, one per agent. Each 
   sprite sheet has 3 poses: idle, working, and walking. The agents are:
   - Space Monkey 🐒 — Mission Director (violet/purple theme)
   - Life Support Officer 🥷🏽 — Infrastructure Controller (pink theme)
   - Systems Engineer 🔧 — Coder/Builder (cyan/teal theme)
   - Station Archivist 📚 — Memory & Knowledge (amber/gold theme)

   Replace all existing sprite placeholders with these new sprites. 
   Each agent should display:
   - Idle pose when at their station with no active tasks
   - Working pose when actively working on a task
   - Walking pose when moving between rooms

2. AGENT OFFICE / SPACESTATION VIEW
   A living 2D pixel-art view showing 4 connected rooms (Command HQ, 
   Security Bay, Engineering, Archive) in a continuous floorplan with 
   no gaps between rooms. Each agent occupies their home room:
   - Space Monkey → Command HQ
   - Life Support Officer → Security Bay
   - Systems Engineer → Engineering
   - Station Archivist → Archive

   The view should feel like looking down on a living settlement — 
   like Age of Empires, but for a space station. Agents are always 
   doing something, even when idle.

3. CREW VISITS — Agents periodically walk across the hub corridor 
   to visit another room. Show EN ROUTE / VISITING labels and path 
   dots along their walking route. When an agent visits another room, 
   their static sprite hides so there's no duplicate.

4. STATION MOOD LIGHTING — A subtle pulsing tint wraps the entire 
   floorplan. The colour and speed shift based on station state:
   - Calm (all systems nominal): slow violet pulse
   - Busy (tasks in progress): moderate amber pulse
   - Alert (incidents open): faster red pulse
   - Critical (P1 incidents): urgent red pulse

5. CELEBRATION OVERLAY — Confetti + sweep flash animation that fires 
   when all tasks are moved to Done. A small visual reward that makes 
   the station feel alive.

6. EMPTY STATE CAMEOS — Themed holding states for empty panels (no 
   incidents, no tasks, no activity) showing pixel-art cameos of the 
   relevant agent.

- Sprites are PNG files with transparent backgrounds
- The build runs on localhost:3000 via bun
- All data comes from the OpenClaw workspace filesystem — no mock data
- The project lives at ~/.openclaw/workspace/mission-control-dashboard/
